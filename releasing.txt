Steps to release a new version of RPicSim:

1) rake todo
2) rake multispec
3) bundle exec rubocop
4) rake doc
5) Update version number in lib/rpicsim/version.rb
6) git commit -am "Updated version to x.x.x"
7) rake build
8) gem push rpicsim-x.x.x.gem
9) git tag x.x.x
10) git push origin && git push origin --tags
11) Upload new docs.
